# README

## Configuration

* Ruby version
	ruby 3.0.0
* Rails version
	Rails 7.0.8
* CSS
	Bootstrap
* JS Bundler
	esbuild

## SETUP

* Database creation
	```
		rails db:create
	```

* Database initialization
	```
		rails db:migrate db:seed
	```
* Set Dependencies
	```
		bundle
	```

##  Launch
